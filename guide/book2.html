MNRE-SLC-001

2. หน้าที่ Admin และ User

Admin หมายถึง ผู้ดูแลการใช้ระบบ GPPC ของ สป.ทส. ในภาพรวม ปัจจุบัน คือ กองกลาง โดยส่วนข้อมูลข่าวสารและบริการร่วม ได้แก่

  1. ผู้อำนวยการส่วนข้อมูลข่าวสารและบริการร่วม
  2. หัวหน้าฝ่ายบริการร่วม
  3. หัวหน้าฝ่ายข้อมูลข่าวสาร

หน้าที่

  1. กำหนดบัญชีและสิทธิของ User
  2. บันทึกค่าตั้งต้น
  3. ให้คำปรึกษาแนะนำเกี่ยวกับวิธีใช้งานแก่ User
  4. จัดทำรายงานสรุปผลการใช้งานระบบ GPPC ต่อผู้บริหาร


User หมายถึง เจ้าหน้าที่ของแต่ละกอง/ศูนย์/กลุ่ม/สถาบัน ในสังกัด สป.ทส. ที่ได้รับมอบหมายให้นำข้อมูลเข้าสู่ระบบ GPPC ปัจจุบัน คือ เจ้าหน้าที่ที่เข้ารับการอบรมหลักสูตรกฎหมายคุ้มครองข้อมูลส่วนบุคคล ภายใต้โครงการ GPPC ในปีงบประมาณ พ.ศ. 2566 จำนวน 10 คน

หน้าที่

  1. นำข้อมูลของหน่วยงานต้นสังกัด เข้าสู่ระบบ GPPC
  2. แจ้งปัญหาและอุปสรรค (ถ้ามี) ต่อ Admin
  3. จัดการคำขอใช้สิทธิ
  4. ปฏิบัติงานอื่น ๆ ตามที่ได้รับมอบหมาย


ทั้งนี้ Admin และ User ควรศึกษาทบทวนวิธีใช้งานแพลตฟอร์ม GPPC ตามที่ได้อบรมมา จากเอกสารด้านล่างนี้
